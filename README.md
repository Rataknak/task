# SpringMicroservices Project

This is a project demonstrating a microservices architecture with Spring Boot and Docker.

## Deployment

This project can be deployed to AWS EC2. The AWS Free Tier includes options for EC2, allowing you to run a microservice like this for free for a year.

### AWS EC2 Free Tier

The AWS Free Tier includes 750 hours of `t2.micro` or `t3.micro` instances per month for one year. This is enough to run a small application 24/7. For more details, see the [AWS Free Tier](https://aws.amazon.com/free/) page.
